Mobile Development

I started mobile development working on an Ionic application designed to mimic tinder for finding resturants. Since then I have worked on an AR mini golf game as well as an Alexa project for Echo Frames.

Tinder for Restaurants - Gobble

April 2022

This project was my first real UX design work. Our team was tasked with incorporating an API into a mobile application and we decided on using Location. After thinking of problems we faced, like not knowing where we want to eat, we came up with the idea of a restaurant picker. To make it more interesting we wanted to also include a tinder swipe feature. My main job on this project was the design, look, and feel of the App.

Skills Gained

  • Introduction to Ionic
  • Mobile App design
  • First experience as head UX Designer

Alexa Interactions for Echo Frames

November 2022

This project was the largest step out of my comfort zone I have experienced. I had to work with a lot of complicated new programs that were challenging but extremely rewarding. The Alexa project I am currently working on with a team is for a "Smart Toilet" that implements Alexa and IoT principles. The goal is to create the Alexa functionality to have a toilet that can be controlled entirely via Echo Frames.

Skills Gained

  • Alexa Intent design
